# Install
1. `git clone --bare <this-repo> $HOME/.cfg`
1. Add `. ~/.chris` to `.bashrc` or `.zshrc`
1. Use the `config` shell alias to manage dotfiles
1. Use `bin/install` to install common packages
1. Install neovim, others as required
